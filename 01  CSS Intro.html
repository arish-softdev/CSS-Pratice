CSS Introduction

  • CSS(Cascading Style Sheets) is a stylesheet language that describes the presentation of HTML elements. It allows us to control layout, colors, fonts, and other vistual aspects of web pages.


1. What is CSS?

  • Separates content (HTML) from presentation (styles).
  • Enables consistent, reusable designs across multiple pages.
  • Supports responsive layouts for different device.

2. How to Add CSS?

  • Inline: Use the style attribute on HTML elements.
  • Internal: Place style rules within the documents head.
  • External: Link to a.css file via link tag.

3. Why use CSS?

  • Maintainability : Keep style separate from maekup.
  • Perfomance : Leverage browser coching for external CSS files.
  • Flexibility : Apply dynamic themes and adapt to new design requriments.

4. When & Timeline

CSS was first proposed in 1994 by HAkon Wium Lie and Bert Bos.Key milestones :

  • 1996 : CSS1 official recommendation by W3C.
  • 1998 : CSS2 with media types and positioning.
  • 2011-2020 : CSS3 modular approach (Flexbox, Grid).
  • 2025 : Latest modules like Container Queries and @layer.

5. CSS Syntax & Structure

A CSS rule consists of a selector and a declaration block.

selector { property: value;}

  • Selectors : Target HTML elements (e.g.,p,.class,#id).
  • Properties : Define the style attribute (e.g.,color,margin).
  • Value : Assigned to properties (e.g.,red,10px).

6. Cascade & Specificity

  • Cascade : Order of style rules, inline > internal > external.
  • Specificity : Weight assigned to selectors (IDs > classes > elements).
  • Inheritance : Some properties inherit from parent elements.

7. CSS Box Model

  • Content : The actual text or media.
  • Padding : Space inside the element.
  • Border : Wraps the Padding.
  • Margin : Space outside the border.
