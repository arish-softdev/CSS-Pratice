CSS Margins


Introduction:

  • CSS Margins are used to create space around elements, outside of any defined borders.
  • They play a vital role in controlling the layout and spacing between elements on a web page.

Key CSS Margin Properties:

  • margin-top: Sets the top margin of an element.
  • margin-right: Sets the right margin of an element.
  • margin-bottom: Sets the bottom margin of an element.
  • margin-left: Sets the left margin of an element.
  • margin: A shorthand property to set all four margins at once.

Step 1: Understanding Margin Property

  • The margin property in CSS is used to control the spacing outside an element.

Step 2: Applying Margins

The margin property can be applied in several ways:
  • margin: value;: Applies the same margin to all sides of the element.
  • margin: top right bottom left;: Applies different margins to each side of the element in clockwise order.
  • margin-top: value;: Applies a specific margin to the top of the element.
  • margin-bottom: value;: Applies a specific margin to the bottom of the element.
  • margin-left: value;: Applies a specific margin to the left of the element.
  • margin-right: value;: Applies a specific margin to the right of the element.
  • margin: vertical horizontal;: Applies different top/bottom and left/right margins.
  • margin: auto;: Centers the element horizontally by setting left and right margins to auto.
