CSS Tables


Introduction:

  • CSS Tables are used to style and format HTML tables, enhancing their appearance and functionality.
  • They provide control over table layout, spacing, borders, colors, and responsiveness.

Key CSS Table Properties:

  • border: Defines the style, width, and color of table borders.
  • padding: Controls the space between cell content and its border.
  • text-align: Sets the horizontal alignment of text within cells.
  • vertical-align: Controls the vertical alignment of content within cells.
  • width / height: Sets the dimensions of the table or individual cells.
  • border-collapse>: Determines whether table borders are collapsed into a single border or separated.
  • background-color: Applies background color to the table or specific cells.

1. Basic Table Styling:

  • Use the table element to define a table.
  • Apply styles to the th and td elements for table headers and data cells, respectively.
  • Use the border-collapse property to collapse the table borders.

2. Alternate Row Styling:

  • Use the :nth-child> selector to target specific table rows.
  • Apply a background color to the even or odd rows to create an alternating row styling effect.

3. Highlighting Specific Cells:

  • Use CSS classes to target specific table cells.
  • Apply a different background color or other styles to highlight those cells.

4. Styling Specific Cells and Columns:

  • Use the :nth-child> selector to target specific cells or columns.
  • Apply custom styles to those cells or columns using CSS properties.

5. Applying Multiple Styles:

  • Combine multiple CSS properties and selectors to apply various styles to specific table cells.
