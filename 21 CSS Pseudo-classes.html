CSS Pseudo-classes and Pseudo-elements


Introduction:

  • Pseudo-classes are used to define the special state of an element (e.g., hover, focus, first-child).
  • Pseudo-elements are used to style specific parts of an element (e.g., before, after).

CSS Pseudo-classes:

  • :hover: Targets an element when it's being hovered over by a user.
  • :focus: Targets an element when it gains focus (e.g., an input field).
  • :first-child: Targets the first child element within a parent element.
  • :first-child: Targets the first child element within a parent element.
  • :not: Targets elements that do not match a given selector.

  • ::before: Inserts content before the content of an element.
  • ::after: Inserts content after the content of an element.
  • ::first-letter: Targets the first letter of an element.
  • ::first-line: Targets the first line of a block-level element.

  • :first-child: In the given HTML and CSS example, the :first-child pseudo-class is used to target the first child paragraph element within the `body` element.
  • When applied, it styles the very first paragraph differently from the rest of the paragraphs in the container.

  • This pseudo-class is useful for scenarios where you want to style the initial element within a container differently.

7. Pseudo-classes: nth-child

  • :nth-child: This pseudo-class targets a specific child within a parent based on its order.
  • It allows you to apply styles to elements based on their position within a parent container.
  • For example, you can use `:nth-child` to select and style the third paragraph within a container or any other specific position.

8. Pseudo-classes: not

  • :not: This pseudo-class targets elements that do not match a given selector.
  • It allows you to select and style elements that do not meet specific criteria.
  • For instance, you can use `:not` to style all paragraphs that are not marked as "special" with a custom class or any other condition that you specify.
