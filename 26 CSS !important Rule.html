CSS !important Rule


Introduction:

  • The `!important` rule in CSS is used to override other style declarations that might apply to the same element.
  • It ensures that a specific CSS declaration takes precedence over others, even if they have higher specificity.

Step 1: Syntax.

The !important rule is used to give a CSS declaration higher specificity, overriding any other styles applied to an element.

selector {

property: value !important;

}

  • Add the !important keyword after the value of the CSS property you want to prioritize.

Example:

This is Blue Color

The !important rule override all paragraph color.
