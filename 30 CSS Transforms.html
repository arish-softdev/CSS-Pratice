CSS 2D/3D Transforms


Introduction:

  • CSS Transforms allow elements to be rotated, scaled, skewed, and translated in 2D or 3D space.
  • They are powerful tools for enhancing visual effects and interactivity on a web page.

CSS 2D Transforms:

  • translate: Moves an element along the X and Y axes.
  • rotate: Rotates an element around a fixed point.
  • scale: Resizes an element.
  • skew: Skews an element along the X and Y axes.

CSS 3D Transforms:

  • translate3d: Moves an element along the X, Y, and Z axes.
  • rotate3d: Rotates an element around a 3D vector.
  • scale3d: Resizes an element in 3D space.
  • perspective: Defines the perspective from which a 3D scene is viewed.
