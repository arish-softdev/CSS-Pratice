CSS Animations


Introduction:

  • CSS Animations enable smooth, customizable animations within web pages.
  • They are defined using keyframes, controlling the intermediate steps of the animation.

Key CSS Animation Properties:

  • @keyframes: Defines the behavior of the animation at various stages or keyframes.
  • animation-name: Specifies the name of the keyframes rule to apply.
  • animation-duration: Controls the total time taken for one cycle of the animation.
  • animation-timing-function: Describes how the intermediate property keyframes are calculated.
  • animation-delay: Sets a delay before the animation starts.
  • animation-iteration-count: Specifies the number of times the animation should repeat.
  • animation-direction: Defines whether the animation should play in reverse or alternate cycles.
