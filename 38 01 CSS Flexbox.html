CSS Flexbox


Introduction:

  • CSS Flexbox, or the Flexible Box Layout, is a layout model that allows easy design of complex layouts with alignment capabilities.
  • It provides an efficient way to distribute space and align items within complex layouts and when the size of your items is unknown or dynamic.

Key Concepts:

  • Flex Container: The parent element that holds flex items.
  • Flex Items: The child elements within a flex container.
  • Main Axis: The primary axis along which flex items are laid out.
  • Cross Axis: The axis perpendicular to the main axis.

Important CSS Flexbox Properties:

  • display: Defines a flex container (e.g., display: flex;).
  • flex-direction: Defines the direction of the main axis.
  • justify-content: Aligns items along the main axis. Horizontal alignment in x-axis.
  • align-items: Aligns items along the cross axis. Vertical alignment in y-axis.
  • align-self: property is used to control the alignment of individual flex items within a flex container.
  • flex-grow, flex-shrink, flex-basis: Control how flex items grow and shrink within the container.
