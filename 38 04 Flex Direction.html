Flex Direction


Understanding "flex-direction" in CSS Flexbox

  • The "flex-direction" property is used to control the direction in which flex items are placed within the flex container.
  • It determines the main and cross axes of the flex container.

The "flex-direction" Property

  1. row:Items are placed the same as the text direction.
  2. row-reverse:Items are placed opposite to the text direction.
  3. column:Items are placed top to bottom.
  4. column-reverse:Items are placed bottom to top.

1. Row Flexbox Layout

  • The `flex-direction: row` property arranges flex items in a horizontal line.
  • Each item has a light blue background with padding and margin, and the container is outlined with a black border.
