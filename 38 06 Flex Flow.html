Flex Flow


What is flex-flow?

The flex-flow property is a CSS Flexbox shorthand that combinesflex-direction andflex-wrap in one declaration. It lets you control both the direction of flex items and whether they wrap onto multiple lines.


Values

  • direction (sets main axis):row, row-reverse, column, column-reverse
  • wrap (controls wrapping):nowrap, wrap, wrap-reverse

1. Row No Wrap (flex-flow: row nowrap;)

Output:
A
B
C

2. Row Wrap (flex-flow: row wrap;)

Output:
1
2
3
4
5

3. Column Wrap (flex-flow: column wrap;)

Output:
X
Y
Z
W

4. Column-Reverse Wrap-Reverse (flex-flow: column-reverse wrap-reverse;")

Output:
@
$
