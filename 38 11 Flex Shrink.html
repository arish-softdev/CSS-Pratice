Flex Shrink


Understanding "flex-shrink" in CSS Flexbox

  • The "flex-shrink" property is used to specify how flex items shrink when there's not enough space.
  • It determines the proportion by which an item shrinks in comparison to other items within the container.

The "flex-shrink" Property

The "flex-shrink" property accepts an integer value representing the shrink factor of the flex item. Items with higher values will shrink more.
