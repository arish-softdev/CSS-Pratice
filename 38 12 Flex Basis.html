Flex Basis


Understanding "flex-basis" in CSS Flexbox

  • The "flex-basis" property is used to define the initial size of a flex item in the main axis.
  • It determines the size of an item before any available space is distributed.

The "flex-basis" Property

The "flex-basis" property accepts length values, such as pixels or percentages, to define the initial size of flex items.
