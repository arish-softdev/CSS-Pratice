CSS Grid Template Rows


Introduction:

  • CSS Grid Template Rows define the rows of a grid container in a flexible and powerful way.
  • They allow for the creation of responsive and dynamic layouts by specifying the size and behavior of rows.

Types of Row Definitions:

  • Fixed-size Rows: Set a specific size for rows (e.g., pixels).
  • Repeat Function: Repeat a pattern of row sizes (e.g., repeat(3, 50px)).
  • Fractional Units (fr): Use fractions to distribute available space (e.g., 1fr 2fr).
  • Auto: Automatically size rows based on content (e.g., auto).
